🎨 主题切换功能说明

功能概述

已成功为主页（index.html）实现了主题切换功能。用户现在可以在浅色和深色主题之间自由切换， 并且主题偏好会自动保存到浏览器本地存储中，下次访问时会自动恢复用户选择的主题。

已实现的功能

  • 浅色主题：基于现有 style.css 的配色方案
  • 深色主题：基于人员画像页面的紫蓝渐变配色
  • 一键切换：点击导航栏中的主题按钮即可切换
  • 自动保存：选择的主题会自动保存到 localStorage
  • 平滑过渡：主题切换带有流畅的动画效果
  • 响应式设计：适配桌面和移动设备

如何使用

在主页（index.html）的右上角导航栏中，点击"🌙 深色"按钮即可切换到深色主题， 按钮会变为"☀️ 浅色"。再次点击可切换回浅色主题。

技术实现

主题切换功能通过 CSS 变量（CSS Variables）和 data-theme 属性实现：

CSS 变量定义位置：
- 浅色主题变量：style.css 第 9-33 行
- 深色主题变量：style.css 第 1197-1221 行

HTML 属性：
- <html data-theme="light"> 浅色主题
- <html data-theme="dark"> 深色主题

扩展到其他页面

复制 index.html 中第 248-332 行的深色主题 CSS 样式到目标页面

步骤 2：在导航栏添加切换按钮
