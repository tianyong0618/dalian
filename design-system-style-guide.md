# 大厂 UI 设计系统 & 全局样式规范手册
## 适用范围
Web官网、SaaS后台、管理系统、移动端App、小程序、数据大屏、高保真原型设计。
可随时指定：遵循 字节/阿里/Apple/Google Material3/腾讯/极简商务/毛玻璃 风格生成界面。

---

# 一、通用基础规范（所有大厂通用）
## 1. 布局与栅格
- 统一采用 12 列网格布局
- 基础间距规格：4px / 8px / 12px / 16px / 24px / 32px / 48px
- 页面左右安全边距：桌面端 24–48px，移动端 16px
- 内容区块必须留白充足，禁止元素拥挤堆砌
- 所有元素严格对齐、视觉均分、层级分明

## 2. 圆角规范
- 小控件：6px
- 常规按钮/输入框：8px
- 卡片容器：12px ~ 16px
- 大模块/弹窗：20px
- 极简风可适当加大圆角，硬核科技风减小圆角

## 3. 阴影规范
- 一级浅阴影：轻微浮起，用于常规卡片
- 二级中阴影：弹窗、下拉菜单、悬浮层
- 三级深阴影：头部导航、底部悬浮按钮
- 拒绝过重死黑阴影，统一用柔和多层阴影

## 4. 字体层级
- 一级标题：24–32px 加粗
- 二级标题：18–22px 半粗
- 三级小标题：16px 常规
- 正文文案：14px
- 辅助小字/标签：12px
- 全局无衬线字体，中英文统一和谐

# 二、各大厂专属设计风格规范

## 1. 字节跳动 极简清冷风
- 主色调：静谧蓝 #1677ff 衍生低饱和色系
- 风格：极简、留白大、线条克制、无多余装饰
- 配色：低饱和、高通透、中性灰偏冷
- 质感：轻卡片、浅阴影、少渐变、干净利落
- 适用：后台系统、工具类产品、官网

## 2. 阿里 Ant Design 商务稳重风
- 主色调：品牌蓝 #006bff
- 风格：规整、严谨、商务、秩序感强
- 布局：模块化分区清晰，表格表单规范标准
- 圆角：适中 8px 为主，不夸张
- 适用：企业SaaS、ERP、后台管理、政务系统

## 3. Apple 苹果极简轻奢风
- 主色调：浅灰基调 + 低饱和点缀色
- 风格：极致留白、大圆角、毛玻璃、通透质感
- 视觉：柔和渐变、轻薄阴影、扁平化无厚重感
- 控件：大圆角按钮、半透明浮层
- 适用：App界面、官网、轻奢产品、展示型页面

## 4. Google Material 3 材质风
- 主色调：动态色彩、同色系深浅渐变
- 风格：层次堆叠、材质感、水面渐变、卡片分离
- 特性：高度圆角、柔和渐变、悬浮抬升效果
- 适用：安卓App、国际化产品、现代化官网

## 5. 腾讯 温润科技风
- 主色调：青蓝 #00b42a 偏温润色系
- 风格：柔和、亲和力强、圆角偏大、过渡自然
- 设计：不尖锐、不冷硬、视觉圆润舒适
- 适用：社交、工具、家庭类、大众产品

## 6. 暗黑科技风 / 暗色模式
- 底色：#0b101c ～ #11141f 深黑蓝
- 主色：霓虹蓝/青绿色高光点缀
- 质感：边框微光、渐变描边、发光文字
- 卡片：半透明黑底、轻微发光阴影
- 适用：数据大屏、AI后台、运维监控、科技产品

## 7. 毛玻璃 Glassmorphism 高级质感
- 特性：backdrop-filter 模糊 + 半透明白底/黑底
- 边框：1px 半透明白色细线
- 背景：渐变底色 + 虚化衬底
- 风格：通透、悬浮、现代感极强
- 适用：首页落地页、仪表盘、App首页

## 8. 极简商务高级风
- 配色：黑白灰 + 单一低饱和主色
- 无花哨渐变、无多余装饰
- 大量留白、居中对齐、层次极简
- 适合：企业官网、个人作品集、高端落地页

# 三、配色系统通用模板
## 基础色板结构
- 主色 Primary
- 辅助色 Secondary
- 中性灰系列（5级深浅）
- 文字色：标题色 / 正文色 / 次要文字色
- 状态色：成功绿 / 警告黄 / 危险红 / 信息蓝

## 设计强制原则
1. 整套页面必须沿用同一套色板，不随意乱换颜色
2. 最多 1 个主色 + 1 个辅助色，其余用中性灰
3. 禁用高饱和刺眼配色，全部低饱和高级灰调
4. 暗色模式保持文字对比度达标，不发灰看不清

# 四、使用说明
1. 将本文件作为全局设计规范参考
2. 用户指定风格时，严格按对应大厂规范输出UI原型
3. 先自动生成匹配风格的色板、字体、圆角、阴影、间距
4. 再按规范输出页面布局 + HTML/Tailwind 高保真代码
5. 全程保持风格统一、视觉高级、层级清晰
