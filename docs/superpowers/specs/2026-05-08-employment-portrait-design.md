# 就业人员画像功能原型改造设计

**日期**：2026-05-08
**基于**：`doc/就业人员画像功能需求说明.docx`
**改造文件**：`原型设计/9-employment-portrait.html`
**方案**：按需求文档结构重组页面（方案A）

---

## 一、概述

### 改造目标
全面按照需求文档改造现有原型页面，实现5大模块：
1. 数据采集与整合
2. 动态标签体系管理
3. 就业人员画像管理（三类画像）
4. AI与技术应用展示
5. 画像应用与动态更新

### 用户权限体系
| 用户类型 | 可见功能 | 数据管理权限 |
|---------|---------|------------|
| 人社部门 | 全部功能 | 完整权限（含安全管理） |
| 经营性人力资源服务机构 | 求职者画像+重点群体画像+标签管理 | 仅查看 |
| 用工企业 | 仅人才画像+岗位推荐 | 无 |

---

## 二、用户类型切换器

**位置**：导航栏下方、页面标题上方
**形式**：3个Radio按钮（人社部门 / 经营性人力资源服务机构 / 用工企业）
**行为**：
- 切换后，页面功能区域按权限高亮/置灰
- 企业用户点击"求职者画像"或"重点群体画像"提示"无权限查看"
- 机构用户可查看求职者画像和重点群体画像，数据管理仅查看
- 人社部门可查看所有类型

---

## 三、数据采集与整合模块

**位置**：核心指标（统计卡片）下方，核心功能入口上方

### 4个数据源卡片
- 社保数据、就业数据、人才数据、维权数据
- 每个卡片包含：数据源名称、采集状态（绿/黄/灰）、最近更新时间、数据量
- 底部显示"数据同步状态：实时同步中" + "进入数据管理"按钮

### 数据管理模态框（独立模态框，4个Tab）
| Tab | 内容 |
|-----|------|
| 采集日志 | 表格：时间、来源、条数、状态 |
| 数据校验 | 校验规则（去重、格式校验、完整性校验）+ 校验结果统计 |
| 安全管理 | 数据加密状态、访问权限设置、操作日志（仅人社部门可见，机构用户隐藏此Tab） |
| 数据清洗 | 清洗规则配置（自动去重、异常值处理） |

---

## 四、动态标签体系管理

### 6大标签分类（按需求文档）
1. **基础信息标签**：年龄、性别、户籍、学历等
2. **教育特征标签**：毕业院校、专业、学历层次、教育竞争力等
3. **就业特征标签**：社保缴纳、就业状态、工作年限、稳定性等
4. **技能标签**：专业技能、证书、语言能力等
5. **帮扶标签**：帮扶等级、帮扶记录、就业困难类型等（新增）
6. **意向标签**：求职意向、薪资期望、工作地点偏好等（新增）

### 标签管理模态框升级
- 左侧：6个分类导航
- 右侧：当前分类的标签列表，支持添加/删除/编辑
- 顶部："AI智能生成标签"按钮
- 底部：标签统计（总数、自动生成数、手动添加数）

### 与画像类型的联动
| 画像类型 | 展示标签 |
|---------|---------|
| 求职者画像 | 教育特征 + 就业特征 + 帮扶标签 + 意向标签 |
| 人才画像 | 教育特征 + 技能标签 + 意向标签 |
| 重点群体画像 | 基础信息 + 就业特征 + 帮扶标签 |

---

## 五、就业人员画像管理

### 人员表格升级
- 新增"画像类型"列（求职者/人才/重点群体），badge颜色：求职者(蓝色)、人才(紫色)、重点群体(橙色)
- 新增"画像类型"筛选下拉框
- 5个示例人员分别归属不同类型

### 画像详情模态框动态切换

**求职者画像**（如李四-就业困难人员）：
- 维度：教育竞争力（88%）、就业稳定性（78%）、帮扶指数（65%）、薪资变化趋势（71%）
- 标签区域：突出显示帮扶标签和意向标签
- 个性化服务：岗位推荐 + 帮扶政策推送

**人才画像**（如张三-高校毕业生）：
- 维度：教育经历（本科/计算机）、职业技能评分（82%）、就业意向匹配度（91%）、薪资预期（15K）
- 标签区域：突出显示技能标签和教育特征标签
- 企业查看视角：仅显示人才能力相关，隐藏敏感帮扶信息

**重点群体画像**（如王五-农民工）：
- 维度：群体规模（28,345人）、群体就业率（88.7%）、主要需求（欠薪维权、技能培训）
- 群体趋势分析：用进度条展示就业率、技能提升率、帮扶覆盖率等指标
- 针对性帮扶策略推荐

### 用户权限拦截
- 企业用户点击"人才画像"可查看，点击其他类型提示"无权限查看"
- 人社部门可查看所有类型
- 机构可查看求职者画像和重点群体画像

## 六、AI与技术应用展示（分散嵌入）

### AI技术标签嵌入各模块
- 数据采集卡片：角标"AI自动采集"
- 标签管理：自动生成的标签带"AI"小图标，注明"由知识图谱分析生成"
- 一键生成画像：步骤2改为"AI数据采集与分析"，步骤3展示"知识图谱关联分析中..."
- 岗位推荐卡片：角标"AI精准匹配 XX%"

### 重点群体监测升级

- "状态"列区分：监测中（绿色）、预警（黄色）、已介入帮扶（蓝色）
