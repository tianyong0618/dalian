## 一、改造目标

全面按照"就业人员画像"的模块结构重构政策计算器，实现5大模块：

1. 用户权限体系（角色切换）
2. 数据采集与整合（政策数据源）
3. 动态标签体系管理（政策标签分类）
4. 核心功能入口（政策测算、文件库、服务事项等）
5. 测算历史与效果追踪（历史记录、政策覆盖率）

---

## 二、用户权限体系

### 2.1 用户类型切换器

**位置**：导航栏右侧（与就业人员画像一致）  
**形式**：下拉选择框（人社部门 / 机构 / 企业 / 个人）  
**行为**：
- 切换后，功能区域按权限高亮/置灰
- 企业/个人用户无法访问政策库管理、数据管理等
- 人社部门可查看全部功能
- 机构可查看政策文件库、测算功能

### 2.2 权限拦截规则

| 功能 | 人社部门 | 机构 | 企业 | 个人 |
|------|---------|------|------|------|
| 数据采集模块 | 完整 | 仅查看 | 无 | 无 |
| 政策文件库 | 完整 | 仅查看 | 仅查看 | 仅查看 |
| 服务事项库 | 完整 | 仅查看 | 仅查看 | 仅查看 |
| 智能政策测算 | 完整 | 完整 | 完整 | 完整 |
| 效果追踪评估 | 完整 | 仅查看 | 无 | 无 |
| 评估报告生成 | 完整 | 仅查看 | 无 | 无 |

## 三、数据采集与整合模块

### 3.1 政策数据源卡片（4个）

**位置**：核心指标下方，核心功能入口上方

| 数据源 | 图标 | 状态 | 内容 |
|--------|------|------|------|
| 政策文件库 | 📜 | 实时同步 | 文件数量、更新日期、数据量 |
| 政策法规库 | ⚖️ | 同步中 | 法规数量、最新法规、关联政策 |
| 政策解读库 | 📖 | 实时同步 | 解读文章、专家解读、图解 |
| 典型案例库 | 📊 | 实时同步 | 成功案例、申领流程、注意事项 |

**卡片样式**：参考就业人员画像的 `data-source-card` 样式，每个卡片包含：
- 数据源名称（顶部，15px字体，加粗）
- 采集状态badge（绿=实时同步，黄=同步中，灰=需手动更新），11px字体
- 最近更新时间（text-tertiary，12px）
- 数据量统计（text-tertiary，12px）
- 底部AI角标：`🤖 AI自动采集`，11px，primary颜色

### 3.2 数据管理入口条

**样式**：与就业人员画像一致
- 背景：rgba(22, 119, 255, 0.05)
- 边框：1px solid rgba(22, 119, 255, 0.2)
- 圆角：var(--radius-md)
- 左侧：数据同步状态文字
- 右侧："进入数据管理"按钮

### 3.3 数据管理模态框

**位置**：点击"进入数据管理"按钮弹出  
**宽度**：800px  
**结构**：4个Tab标签

| Tab | 内容 |
|-----|------|
| 采集日志 | 表格：时间、来源、条数、状态 |
| 数据校验 | 校验规则配置 + 校验结果统计 |
| 安全管理 | 数据加密状态、访问权限、操作日志（仅人社可见） |
| 数据清洗 | 清洗规则配置（自动去重、异常值处理） |

## 四、动态标签体系管理

### 4.1 政策标签分类（6大类）

采用与就业人员画像一致的标签分类思路：

1. **补贴类型标签**：就业补贴、培训资助、创业扶持、社保减免、稳岗返还
2. **适用对象标签**：高校毕业生、就业困难人员、退役军人、农民工、企业
3. **申领条件标签**：社保缴纳、劳动合同、收入要求、企业规模
4. **优惠力度标签**：补贴金额、减免比例、贷款额度
5. **有效期限标签**：长期有效、阶段性政策、即将到期
6. **区域限定标签**：全市通用、区域限定、试点政策

### 4.2 标签管理模态框

**宽度**：900px  
**结构**：左右布局

- **左侧**：6个分类导航（垂直菜单），宽度200px
  - 每个分类名称 + 标签数量
  - 选中状态：primary颜色背景
- **右侧**：当前分类的标签列表
  - 每个标签：名称 + 使用次数 + 操作（编辑/删除）
  - 支持搜索过滤
- **顶部**：`🤖 AI智能生成标签` 按钮，primary颜色
- **底部**：标签统计（总数、AI生成数、手动添加数）

## 十二、技术实现要点

### 12.1 用户切换实现

```javascript
// 用户类型
const userRoles = ['gov', 'org', 'enterprise', 'personal'];

// 切换逻辑
function switchRole(role) {
  // 隐藏/显示对应区域
  document.querySelectorAll('.gov-only').forEach(el => {
    el.style.display = role === 'gov' ? 'block' : 'none';
  });
  // 更新权限状态
  updatePermissionUI(role);
}
```

### 12.2 数据源卡片组件
