## 一、测试概述

### 1.1 测试范围

本次测试基于修复后的调解仲裁服务系统原型进行全面测试，涵盖五类用户角色和核心业务流程。测试重点验证：

1. **劳动者端**：案件申请（含AI材料校验）、合同签署（详情正确显示）、文书签收、云庭审参与（五分区完整页面、全屏、ESC键关闭、主题适配）、法律援助、主题切换
2. **用人单位端**：合同签署（发起签署正常、新建合同列表更新）、工资确认、欠薪投诉处理、主题切换
3. **仲裁机构端**：案件管理（新建案件、筛选）、调解管理（分配案件逻辑正确）、云庭审五分区（全屏、ESC关闭、主题适配）、文书送达、案件分流跳转、AI智能校验（无查看详情按钮）、上下联动详情样式、主题切换
4. **人社监管端**：项目监管、AI欠薪预警（样式正确）、主题切换
5. **法律援助端**：援助申请审核、律师指派、主题切换
