# 人力资源服务旗舰店系统 - 核心功能与测试用例（最终版）

&gt; 文档生成日期：2026-05-19  
&gt; 基于原型版本：12-hr-flagship.html（单页面应用版本）  
&gt; 测试视角：原型演示系统验收

---

## 一、系统概述

### 1.1 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    人力资源服务旗舰店系统                      │
├─────────────────────────────────────────────────────────────┤
│  顶部导航栏                                                    │
│  ├── Logo区域：人社智能服务系统                                 │
│  ├── 角色切换：人社部门 / 服务机构 / 用工企业                    │
│  └── 主题切换：深色/浅色模式                                    │
├─────────────────────────────────────────────────────────────┤
│  Tab标签栏（根据角色权限动态显示）                              │
│  ├── 🏢 机构管理      [gov]                                   │
│  ├── 👥 从业人员管理   [gov, org]                             │
│  ├── 🛒 线上服务网店   [gov, org, enterprise]                 │
│  ├── 🤝 供需撮合      [gov, org, enterprise]                  │
│  ├── ⚖️ 争议调解      [gov, org, enterprise]                  │
│  └── ⭐ 诚信评价      [gov]                                   │
├─────────────────────────────────────────────────────────────┤
│  内容区域（单页面动态切换，无iframe）                           │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 角色权限矩阵

| 功能模块 | 人社部门(gov) | 服务机构(org) | 用工企业(enterprise) |
|---------|--------------|--------------|---------------------|
| 机构管理 | ✅ | ❌ | ❌ |
| 从业人员管理 | ✅ | ✅ | ❌ |
| 线上服务网店 | ✅ | ✅ | ✅ |
| 供需撮合 | ✅ | ✅ | ✅ |
| 争议调解 | ✅ | ✅ | ✅ |
| 诚信评价 | ✅ | ❌ | ❌ |

### 1.3 系统特点

- **单页面应用(SPA)**：无需iframe，页面切换流畅
- **深色科技风**：采用深色背景、紫色主色调的现代化设计
- **响应式设计**：支持移动端、平板端、桌面端适配
- **角色权限控制**：Tab根据角色动态显示/隐藏
- **主题切换**：支持深色/浅色模式切换，状态保持

## 二、各角色功能模块

### 2.1 人社部门(gov) - 6大模块

| 模块 | 功能描述 | 核心功能点 |
|-----|---------|----------|
| 机构管理 | 机构准入、日常监管、运营监测 | 统计看板、机构列表、搜索筛选、新增机构 |
| 从业人员管理 | 从业人员备案、资质管理、培训管理 | 统计看板、人员列表、搜索筛选、批量审核 |
| 线上服务网店监管 | 服务产品审核、网店监管 | 统计看板、服务列表、审核操作、强制下架 |
| 供需撮合 | 企业需求管理、撮合活动管理 | 统计看板、需求列表、推荐机构、活动管理 |
| 争议调解 | 争议受理、调解处理、协议签署 | 统计看板、争议列表、受理操作、调解处理 |
| 诚信评价 | 机构信用评级、星级管理 | 统计看板、信用列表、评分调整、评定历史 |

### 2.2 服务机构(org) - 6大模块

| 模块 | 功能描述 | 核心功能点 |
|-----|---------|----------|
| 机构信息管理 | 机构基本信息、资质管理 | 统计展示、基本信息查看、编辑功能 |
| 从业人员管理 | 本机构人员管理、资质备案 | 统计看板、人员列表、新增人员、编辑人员 |
| 线上服务网店 | 服务产品管理、订单管理 | 统计看板、服务列表、上架/下架、编辑服务 |
| 供需撮合 | 商机查看、投标管理 | 统计看板、需求列表、查看详情、投标操作 |
| 争议处理 | 争议查看、回应调解 | 统计看板、争议列表、查看详情、回应调解 |
| 诚信评价 | 本机构信用查看、评定历史 | 统计展示、评分构成、评定历史 |

### 2.3 用工企业(enterprise) - 4大模块

| 模块 | 功能描述 | 核心功能点 |
|-----|---------|----------|
| 服务商城 | 浏览服务、选购服务 | 统计看板、服务网格、搜索筛选、查看详情、订购 |
| 用工需求 | 发布需求、查看匹配 | 统计看板、需求列表、发布需求、查看匹配、取消需求 |
| 订单管理 | 订单查看、进度跟踪 | 统计看板、订单列表、状态筛选、确认订单、评价 |
| 评价与维权 | 服务评价、维权申请 | 统计看板、评价列表、去评价、申请维权、查看进度 |

- **测试类型**：基础功能测试
- **优先级**：P0
- **测试步骤**：

  - ❌ FAIL：关键元素缺失
