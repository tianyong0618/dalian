# 调解仲裁服务系统 - 核心功能与测试用例

> **文档版本**: v1.0
> **创建日期**: 2026-05-15
> **涉及页面**: 14-mediation-arbitration.html (五角色统一入口)

---

## 一、系统概述

调解仲裁服务系统是劳动人事争议"无接触"调解仲裁服务平台，围绕**五类用户角色**、**七大功能模块**、**完整云庭审五分区**和**AI智能可视化**四大核心构建。

### 1.1 五类用户角色

| 角色 | 文件 | 核心职责 |
|------|------|----------|
| 劳动者端 | `14c-laborer.html` | 申请仲裁、签署合同、查看案件、参与云庭审 |
| 用人单位端 | `14b-employer.html` | 合同管理、考勤上报、应诉答辩、工资确认 |
| 仲裁机构端 | `14a-arbitration.html` | 案件审核、调解管理、云庭审主持、文书送达 |
| 人社监管端 | `14e-supervisor.html` | 项目监管、欠薪预警、效能统计、报表导出 |
| 法律援助端 | `14d-legal-aid.html` | 援助申请审核、律师指派、服务统计 |

### 1.2 七大功能模块

```

│                    七大功能模块                              │

│  电子劳动   │  根治欠薪   │  互联网+    │  在线劳动人事     │
```

## 二、核心业务流程

### 2.1 完整业务闭环

```
申请立案 ──→ 审核受理 ──→ 调解/仲裁 ──→ 云庭审 ──→ 文书送达 ──→ 归档
    │            │             │            │           │          │
 [在线提交]   [AI校验]    [调解员分配]   [五分区]   [云送达]   [数据同步]
```

## 三、核心功能详解

### 3.1 电子劳动合同管理

| 功能点 | 说明 | 涉及角色 |
|--------|------|----------|
| 合同模板选择 | 固定期限/无固定期限/劳务派遣/灵活用工等 | 劳动者、用人单位 |
| 电子签章 | 双方在线签署，带时间戳认证 | 劳动者、用人单位 |
| 合同备案 | 签署完成后自动同步至根治欠薪平台 | 用人单位 |
| 合同查询 | 查看历史合同、合同详情、下载 | 劳动者、用人单位 |

### 3.2 根治欠薪一体化平台

| 功能点 | 说明 | 涉及角色 |
|--------|------|----------|
| 考勤数据上报 | 工程项目农民工考勤记录上传 | 用人单位 |
| 工资发放确认 | 工资确认农民工端确认 | 用人单位、劳动者 |
| 欠薪投诉响应 | 受理/处理/结案欠薪投诉 | 用人单位 |
| 欠薪预警 | AI识别欠薪风险，推送预警 | 人社监管端 |

### 3.3 互联网+云仲裁服务

| 功能点 | 说明 | 涉及角色 |
|--------|------|----------|
| 案件申请 | 填写申请信息、上传证据材料 | 劳动者 |
| AI材料校验 | 实时校验身份证/合同/工资流水/考勤等 | 劳动者 |
| 案件分流 | AI推荐调解优先或仲裁处理 | 仲裁机构 |
| 云庭审 | 五分区完整庭审（视频/证据/笔录/操作） | 仲裁机构、劳动者、用人单位 |
| 云送达 | 裁决书/调解书在线送达确认 | 仲裁机构、劳动者 |

### 3.4 在线劳动人事争议调解

| 功能点 | 说明 | 涉及角色 |
|--------|------|----------|
| 调解申请 | 提交调解申请、证据材料 | 劳动者 |
| 调解员分配 | AI智能匹配或手动选择调解员 | 仲裁机构 |
| 线上调解 | 双方在线沟通、调解协议电子签章 | 劳动者、用人单位、仲裁机构 |

### 3.5 数字化仲裁庭（云庭审五分区）

| 区域 | 功能 | 操作按钮 |
|------|------|----------|
| **信息栏** | 案件编号、类型、状态、开庭时间、倒计时 | 退出 |
| **视频会议区** | 仲裁员/申请人/被申请人画面 | 静音、摄像头 |
| **证据展示区** | 证据列表、实时共享、质证标记 | 上传、标记、展示 |
| **庭审笔录区** | 语音转文字、实时笔录、历史记录 | 导出、历史、确认签字 |
| **操作栏** | 上传证据、举证质证、补充发言、休庭 | 电子签字 |

### 3.6 仲裁工作上下联动

| 功能点 | 说明 | 涉及角色 |
|--------|------|----------|
| 跨区域案件协同 | 案件流转、数据共享 | 仲裁机构 |
| 数据同步状态 | 同步进度监控 | 仲裁机构 |
| 效能统计 | 各级机构办结率、云庭审使用率 | 人社监管端 |

### 3.7 AI功能可视化

| 功能 | 展示方式 |
|------|----------|
| 材料校验 | ✓(绿)/⚠(黄)/✗(红) + 详细提示 |
| 案件分流 | 推荐结果+理由+备选方案弹窗 |
| 欠薪预警 | 红色悬浮卡片+级别标识(红/黄)+推送按钮 |
| 智能分配 | 调解员匹配度评分+工作负载显示 |
