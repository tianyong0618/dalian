## 一、系统概述

人力资源服务旗舰店系统是一个面向人社部门、服务机构和用工企业的综合性数字化服务平台。系统采用SPA（单页应用）架构，通过Tab标签页切换不同功能模块，支持三种角色视角：人社部门、服务机构、用工企业。

### 1.1 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    人力资源服务旗舰店系统                      │
├─────────────────────────────────────────────────────────────┤
│  顶部导航栏                                                    │
│  ├── Logo区域：人社智能服务系统                                 │
│  ├── 角色切换：人社部门 / 服务机构 / 用工企业                    │
│  └── 主题切换：深色/浅色模式                                    │
├─────────────────────────────────────────────────────────────┤
│  Tab标签栏（根据角色权限动态显示）                              │
│  ├── 🏢 机构管理      [gov]                                   │
│  ├── 👥 从业人员管理   [gov, org]                             │
│  ├── 🛒 线上服务网店   [gov, org, enterprise]                 │
│  ├── 🤝 供需撮合      [gov, org, enterprise]                  │

└─────────────────────────────────────────────────────────────┘
```

## 二、核心功能模块梳理

### 2.1 机构管理模块（12a-org-management.html）

#### 2.1.1 功能概述
机构管理模块是人社部门对人力资源服务机构进行准入审核、日常监管和运营监测的核心功能。

#### 2.1.2 核心功能点

| 功能分类 | 功能点 | 功能描述 |
|---------|-------|---------|
| **机构准入与备案** | 机构注册 | 在线提交注册申请，上传资质材料，填写基本信息 |
| | 资质审核 | 人社部门在线审核，留痕可追溯，支持退回修改 |
| | 备案管理 | 自动生成备案编号，支持变更申请与状态更新 |
| **机构日常管理** | 机构列表 | 展示入驻机构信息，支持搜索、筛选、查看详情 |
| | AI运营监测 | 实时采集服务数据，智能分析运营状态，自动预警异常 |
| | 统计看板 | 入驻机构数、平均合规率、本月预警、年度审核待办 |

#### 2.1.3 关键数据字段

- 机构名称、服务类型（劳务派遣/招聘外包/高端猎头/培训服务）
- 星级评分（1-5星）、本月撮合数量、满意度百分比
- 运营状态（营业中/审核中/已暂停）

---

### 2.2 从业人员管理模块（12b-employee-management.html）

#### 2.2.1 功能概述
从业人员管理模块实现人力资源服务行业从业人员的全生命周期管理，从信息采集到诚信档案建立。

#### 2.2.2 核心功能点

| 功能分类 | 功能点 | 功能描述 |
|---------|-------|---------|
| **信息入库** | 信息采集 | 批量导入/单个添加，采集基本信息、从业资质、经历等 |
| | AI信息核验 | OCR识别+AI比对，自动校验身份证、资质证书真实性 |
| | 入库审核 | 人社部门在线审核，生成唯一入库编号 |
| | 信息更新 | 机构在线更新从业人员信息，实时同步 |
| **全生命周期管理** | 培训管理 | AI推荐培训课程，记录时长与考核结果（年培训≥10课时） |
| | 岗位管理 | 记录岗位分配变动，关联机构服务项目 |
| | 离职管理 | 标记离职状态，保留历史记录，支持重新入库 |
| | 诚信档案 | 记录从业行为，关联机构诚信评级，违规者限制入库 |

#### 2.2.3 关键数据字段

- 姓名、身份证号（脱敏显示）、所属机构
- 从业资质（人力资源从业资格证/高级人力资源管理师等）
- 培训课时、在职状态

---

### 2.3 线上服务网店模块（12c-online-store.html）

#### 2.3.1 功能概述
线上服务网店模块为优质人力资源服务机构提供在线展示、服务交付和交易管理的数字化平台。

#### 2.3.2 核心功能点

| 功能分类 | 功能点 | 功能描述 |
|---------|-------|---------|
| **网店开设与管理** | 网店申请 | 优质机构申请开设网店，提交服务范围、收费标准等 |
| | 网店审核 | 人社部门审核，规范服务范围，禁止超资质服务 |
| | 网店运营 | 维护网店信息、发布动态、回复咨询、管理订单 |
| | 网店监管 | 实时监测运营，处理违规，暂停/关闭违规网店 |
| **核心服务功能** | 岗位发布与管理 | 企业发布岗位，AI合规校验，机构筛选对接，智能推送人才 |
| | 视频面试服务 | AI生成面试题库，记录面试过程，生成评价报告 |
| | 政策咨询服务 | AI智能客服7×24小时在线，智能推送适配政策 |

#### 2.3.3 服务流程

```
需求对接 → 人才匹配 → 面试录用 → 签约入职
```

### 2.4 供需撮合模块（12d-demand-matching.html）

#### 2.4.1 功能概述
供需撮合模块通过AI智能算法实现企业用工需求与人力资源服务机构供给的精准匹配。

#### 2.4.2 核心功能点

| 功能分类 | 功能点 | 功能描述 |
|---------|-------|---------|
| **AI智能撮合** | 需求采集分析 | 构建企业需求画像与机构供给画像，多维度解析匹配要点 |
| | 智能匹配推送 | 标注匹配度评分与理由，向双方推送适配对象 |
| | 撮合跟踪 | 记录对接进度，分析未合作原因，推送优化建议 |
| | 人才库检索 | 统一人才库，关键词筛选，AI智能推荐 |
| **线下对接** | 洽谈会组织 | 在线发起，设置主题时间地点，在线报名审核 |
| | 参会匹配 | AI提前匹配对接对象，生成对接清单 |
| | 专场对接 | 针对重点企业/行业发起专场，在线报名参与 |

---

### 2.5 争议调解模块（12e-dispute-mediation.html）

#### 2.5.1 功能概述
争议调解模块提供在线劳动争议申请、受理、调解全流程服务，实现争议的快速化解。

#### 2.5.2 核心功能点

| 功能分类 | 功能点 | 功能描述 |
|---------|-------|---------|
| **申请与受理** | 争议申请 | 在线提交申请，填写详情，上传证据，选择调解方式 |
| | 申请审核 | 人社部门审核真实性，驳回不符申请并说明理由 |
| | 调解员分配 | AI根据争议类型、所在地智能分配调解员 |
| **线上调解流程** | 调解通知 | 自动发送通知，告知调解时间、入口 |
| | 线上调解 | 视频/文字调解，上传补充证据，全程记录 |
| | 调解协议 | 在线签署电子协议，具备法律效力 |
| | 调解归档 | 自动归档记录、协议、证据，支持查询 |

---

### 2.6 诚信评价模块（12f-credit-evaluation.html）

#### 2.6.1 功能概述
诚信评价模块建立人力资源服务机构的信用评级体系，通过量化评分实现行业自律和优胜劣汰。

#### 2.6.2 核心功能点

| 功能分类 | 功能点 | 功能描述 |
|---------|-------|---------|
| **评价体系** | 评价指标 | 合规运营、服务质量、争议处理、稳就业贡献等 |
| | AI自动评分 | 实时采集数据，自动计算评分，每月更新 |
| | 评价等级 | 优秀/良好/合格/不合格，挂钩服务权益 |
| **星级推介** | 星级展示 | 首页、网店显著展示星级标识，优先推荐给企业 |
| | 动态管理 | 定期复核，评分升降则调整星级，激励优质机构 |

#### 2.6.3 星级划分标准

| 星级 | 分值范围 | 评定标准 |
|-----|---------|---------|
| ★★★★★ 五星级 | 240分及以上 | 服务优质、诚信合规、稳就业贡献突出 |
| ★★★★☆ 四星级 | 190-239分 | 服务良好、诚信合规、稳就业贡献显著 |
| ★★★☆☆ 三星级 | 130-189分 | 服务合格、诚信合规、有一定稳就业贡献 |

## 三、演示流程设计

```
【准备】选择角色：人社部门

      ├── 查看机构列表（搜索、筛选功能）

步骤4：切换至诚信评价模块
      ├── 查看星级分布统计
```
